# 项目说明

## 项目目标
构建一个 Python 程序，用于处理包含 232 个反射峰的光谱/距离域数据，并完成以下任务：
1. 自动检测反射峰
2. 自动截取每个单峰
3. 对每个单峰执行 IFT
4. 为每个峰生成双图：
   - 左图：原始全局峰图，并高亮当前峰
   - 右图：当前峰的 IFT 结果图
5. 支持自动播放所有结果
6. 支持导出 GIF 或 MP4

## 开发约束
- 仅使用 Python
- 优先保证可读性和模块化
- 不要把全部逻辑写进一个文件
- 每次只实现一个模块或一个小目标
- 在做较大改动前，先说明计划
- 除非我明确允许，否则不要引入重量级依赖
- 代码要适合新手阅读，函数和关键步骤加注释
- 优先生成可测试、可运行的最小版本（MVP）
- 如果某处不确定，不要假装确定，请明确说明

## 输出要求
每次完成任务后，请：
1. 说明你修改了哪些文件
2. 说明每个文件的作用
3. 说明我下一步该测试什么
4. 如果有风险点或未完成项，请单独列出
